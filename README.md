# mediatracker
sort and track your media
## installation
```
git clone https://github.com/Hauptling12/mediatracker
cd mediatracker
mkdir ~/.local/share/mediatracker/
cp date.json ~/.local/share/mediatracker/
cp mediatracker.py ~/.local/bin
```
## man page
+ `-a` add media
+ `-l` list media
+ `-h` show the man page
